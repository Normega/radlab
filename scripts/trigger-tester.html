Breath Belt — COM trigger tester

Chrome/Edge only. Drives a Black Box ToolKit USB TTL Module over Web Serial: RR resets/clears all lines, a 2-char hex string sets the 8 TTL output lines (e.g. 0A = value 10), 00 clears. The module feeds the ADInstruments PowerLab digital input. No build step — just open and click.
